# Employee Attrition Prediction – ML Classification Pipeline

## Overview
Built a machine learning pipeline to predict employee attrition 
using IBM HR Analytics dataset (1,470 records).

## Tools Used
Python, Pandas, Scikit-learn, Decision Tree, SMOTE, AUC-ROC, Power BI

## Dataset
IBM HR Employee Attrition dataset — 1,470 records, 35 features

## Key Findings
- Identified top attrition drivers: overtime, job level, monthly income
- Handled class imbalance using SMOTE
- Decision Tree model selected for interpretability
- Visualized attrition patterns in Power BI dashboard

## Files
- `employee_attrition.ipynb` — Full ML pipeline
- `WA_Fn-UseC_-HR-Employee-Attrition.csv` — Dataset
- `Employee_Attrition_Dashboard.pbix` — Power BI dashboard
